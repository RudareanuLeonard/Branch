# Branch

An interactive terminal file explorer written in Rust.

Branch provides fast filesystem navigation, fuzzy search, file previews, and directory size analysis directly from the terminal.

Implemented as a learning project focused on Rust systems programming concepts.

## Features

- Recursive directory tree view
- Keyboard-driven navigation
- Fuzzy file and directory search
- File preview panel
- Background directory size calculation
- Hidden file toggle
- Cross-platform terminal interface
